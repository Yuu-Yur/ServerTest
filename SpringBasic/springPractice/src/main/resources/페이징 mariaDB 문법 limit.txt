limit n , m
n개의 행을 skip , m개를 limit
즉 한 페이지에 m개를 보여줌
보여주고 싶은 페이지 page
n = (page - 1)*m
변수는 두개, page 와 m 이 필요
m = size
즉 limit (page - 1)*m , m
하면 page 페이지의 정보를 select 할 수 있음

총 개수 counter
총 페이지 counter/size + 1


select *
from tbl_todo
order by tno
